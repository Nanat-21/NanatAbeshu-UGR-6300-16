## вҡҷпёҸ How It Works

* The app starts with `runApp()` and loads the main widget.
* A `Scaffold` provides the basic structure.
* A `Column` arranges elements vertically.
* A `CircleAvatar` displays the profile image.
* `Text` widgets show name and role.
* `Card` and `Row` are used for contact information.
