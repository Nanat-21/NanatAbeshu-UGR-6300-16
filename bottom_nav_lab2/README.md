# Bottom Navigation Lab

A Flutter application demonstrating bottom navigation bar with screen navigation.

## Learning Objectives

- Use `BottomNavigationBar` to switch between screens
- Manage navigation state with `setState`
- Pass data between screens
- Understand `Navigator.push` and `Navigator.pop`

## Screens

- **Home Screen**: Contains a button that navigates to a details screen
- **Favorites Screen**: Displays a placeholder for favorite items
- **Profile Screen**: Displays user profile information
- **Details Screen**: Shows a back button to return to the home screen

## Features

- Bottom navigation bar with three tabs (Home, Favorites, Profile)
- Navigation stack management using Navigator
- Stateful widget for managing selected index
- Material Design components
